了解Groovy

 2018-04-07   作者: TanDi

介绍

  • apache groovy是一个兼容java语法，面向对象的编程语言
  • 它既是静态语言也是动态语言
  • 可以作为java平台的编程语言和脚本语言，也能编译成jvm字节码，并可以与其他java代码无缝连接操作
  • Groovy支持闭包，多行字符串和嵌入字符串的表达式

特点

一、学习曲线平坦

如果你熟悉java语言或者c#等，那么则可以很快速就上手，因为groovy兼容java语法；

就算你不熟悉java语言和c#，对于groovy的可读和富有表达能力的语法来说，学习起来也显得比较轻松

二、与java集成

与java和java的第三方库无缝集成和相互操作

三、功能强大

支持闭包、构建器、运行时和编译时的元编程、函数式编程、类型推断、静态编译等

四、脚本和测试的小能手

非常适合用来编写简洁和可维护的测试，以及所有的构建、自动化任务等

五、活跃的生态

Web开发，响应式应用程序，并发/异步/并行库，测试框架，构建工具，代码分析，GUI构建

groovy的运行时调度

int method(String arg) {
    return 1;
}
int method(Object arg) {
    return 2;
}
Object o = "Object";
int result = method(o);

// java：静态绑定，编译时根据方法接受的类型判断
2

//groovy：运行时调度，在运行时，通过参数的类型来选择方法
1

内部类

// java 创建非静态内部类
public class Y {
    public class X {}   
    public static void main(String[] args){
        Y y = new Y();
        X x = y.new X();
    }
}

// groovy
public class Y {
    public class X {}

    static main(args) {
        def y = new Y()
        def x = new X(y)
    }
}

lambdas

// java
Runnable run = () -> System.out.println("Run");
list.forEach(System.out::println);

// groovy
Runnable run = { println 'run' }
list.each { println it } // or list.each(this.&println)

}

原始类型和包装类型

以下这段代码，在java中执行，结果是 m(long l)，在groovy结果是void m(Integer i)

原因是，在java中扩展优先于取消装箱，而groovy中所有原始引用都使用它们的包装类

m(i)

  println "in m(Integer)"
}
