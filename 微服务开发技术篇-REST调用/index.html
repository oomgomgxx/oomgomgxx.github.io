Spring RestTemplate

基本介绍

  RestTemplate位于 org.springframework.web 包下，是Spring提供的HTTP请求包装类。默认采用JDK的HttpURLConnection来实现。除了 HttpURLConnection 之外还支持 Netty4、OkHttp 还用 HttpClient。采用那一种实现取决于实现是否存在类路径下。顺序是 HttpClient 、OkHttp 、HttpURLConnection，即如果没有添加 HttpClient 和 OkHttp 的依赖就会采用 HttpURLConnection 来实现。

OpenFeign

基本介绍

  Feign本来是Spring Cloud Netflix开源项目的一部分，后来从项目中独立了出来成为一个通用的Java类库并改名为OpenFeign。OpenFeign是一个声明式、模板化的HTTP客户端，支持Feign、JAX-RS、SpringMVC等风格注解且内置Ribbon和Hystrix。和RestTemplate一样，默认采用URLConnection进行HTTP通信，同时支持Apache HttpClient和OkHttp替换

  • 以上是 SynchronousMethodHandler 的一段代码，可以看到在 Feign 中采用功能开关/模式开关的操作来控制日志的数据
  • Feign只输出 DEBUG 级别的日志，Feign Logger中的 模式Level 有以下几种：
    • NONE：不记录任何日志（默认级别）
    • BASIC：记录请求方法、URL、响应状态码以及执行的时间
    • HEADERS：BASIC + header 信息
    • FULL：记录请求和响应的 header、body、metadata等信息
