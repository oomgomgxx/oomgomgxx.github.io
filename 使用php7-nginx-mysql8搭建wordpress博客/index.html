php7+nginx+mysql8搭建WordPress博客

 2018-02-07   作者: TanDi

说明

  1. 最近打算从hexo迁移到wordpress，虽然hexo比较简便，也很多人说hexo比wordpress好，但是我还是更加喜欢wordpress强大的插件和丰富的主题，而hexo在定制方面略显繁琐，个人并不希望将时间花费在这方面上。

  2. 如果你连wordpress都不想搭建的话，可以跳过这篇文章，直接去阿里云买一个带有wordpress的轻量级服务器即可。

搭建环境

  • CentOS 7
  • nginx
  • php7
  • mysql8
  • wordpress-5.0.3

安装过程

yum upgrade

[root@localhost ~]# yum upgrade

nginx安装

官方安装教程：http://nginx.org/en/linux_packages.html

添加yum源

[root@localhost ~]# vim /etc/yum.repos.d/nginx.repo

# 添加nginx稳定版的yum源
[nginx-stable]
name=nginx stable repo
baseurl=http://nginx.org/packages/centos/7/$basearch/
gpgcheck=1
enabled=1
gpgkey=https://nginx.org/keys/nginx_signing.key

安装nginx

[root@localhost ~]# yum install nginx

禁用当前系统防火墙

禁用防火墙为了减少阻碍，实际可以开放特定端口

[root@localhost ~]# systemctl stop firewalld.service 
[root@localhost ~]# systemctl disable firewalld.service

php7安装

添加 epel 包的仓库源

[root@localhost ~]# yum -y install epel-release

升级/安装 PHP7-FPM webtatic 仓库

[root@localhost ~]# rpm -Uvh https://mirror.webtatic.com/yum/el7/webtatic-release.rpm

安装 PHP7-FPM

[root@localhost ~]# yum -y install php71w-fpm php71w-cli php71w-gd php71w-mcrypt php71w-mysql php71w-pdo php71w-xml php71w-pear php71w-mbstring php71w-mbstring php71w-json php71w-pecl-apcu php71w-pecl-apcu-devel

验证是否安装成功

[root@localhost ~]# php -v

Copyright (c) 1997-2018 The PHP Group
Zend Engine v3.1.0, Copyright (c) 1998-2018 Zend Technologies

安装mysql8

官方安装教程：https://dev.mysql.com/doc/refman/8.0/en/linux-installation-yum-repo.html

安装mysql8源

[root@localhost ~]# yum localinstall https://repo.mysql.com//mysql80-community-release-el7-2.noarch.rpm

查看源是否安装成功

[root@localhost ~]# yum repolist all | grep mysql

安装mysql8

[root@localhost ~]# yum install mysql-community-server

拓展：以下是terminal打印出来的消息，可以看出mysql-community-server包含以下组件

正在安装:
mysql-community-libs

mysql-community-server
mysql-community-client
mysql-community-common

初始化root密码

  1. 首先查看刚才yum安装mysql的安装目录，找到mysql配置文件my.cnf所在的位置
[root@localhost ~]# rpm -ql mysql-community-server 
/etc/logrotate.d/mysql
/etc/my.cnf
/etc/my.cnf.d
。。。省略
  1. 修改my.cnf
[root@localhost ~]# vim /etc/my.cnf

#skip-grant-tables

  1. 启动或重启mysql服务
[root@localhost ~]# systemctl start mysqld.service
  1. 进入mysql初始化root用户密码
[root@localhost ~]# mysql -uroot -p
Enter password: 这里直接回车
mysql> flush privileges;
mysql> use mysql
mysql> alter user 'root'@'localhost' IDENTIFIED with mysql_native_password BY 'My@123456';
mysql> update user set host='%' where user = 'root';

修改密码命令

alter user ‘root’@’localhost’ IDENTIFIED BY ‘你的密码’;

  1. 将第2部添加的配置去除
[root@localhost ~]# vim /etc/my.cnf

  1. 重启mysql服务
[root@localhost ~]# systemctl restart mysqld.service
  1. 使用密码登录mysql
[root@localhost ~]# mysql -uroot -pMy@123456

配置过程

配置 php-fpm 与 Nginx 协同运行

配置php-fpm使用nginx用户来运行

[root@localhost ~]# vim /etc/php-fpm.d/www.conf

配置nginx配置文件

[root@localhost ~]# vim /etc/nginx/conf.d/default.conf

location ~ \.php$ {
    root   /usr/share/nginx/html;
    fastcgi_pass 127.0.0.1:9000;
    fastcgi_index index.php;
    fastcgi_param SCRIPT_FILENAME $document_root$fastcgi_script_name;
    include fastcgi_params;
}

启动/重启nginx服务

[root@localhost conf.d]# systemctl restart nginx.service

启动/重启php-fpm服务

[root@localhost php-fpm]# systemctl start php-fpm.service

查看上面两个服务的应用使用有启动成功

nginx：80端口

php-fpm：9000端口

[root@localhost php-fpm]# netstat -ant | grep 9000
tcp        0      0 127.0.0.1:9000          0.0.0.0:*               LISTEN     
[root@localhost php-fpm]# netstat -ant | grep 80
tcp        0      0 0.0.0.0:80              0.0.0.0:*               LISTEN

验证是否配置nginx和php-fpm

[root@localhost php-fpm]# vim /usr/share/nginx/html/phpinfo.php

添加一下内容

<?php
phpinfo();

出现如画面表示已经配置成功

设置开启启动nginx和php-fpm两个服务

[root@localhost php-fpm]# systemctl enable php-fpm.service 
Created symlink from /etc/systemd/system/multi-user.target.wants/php-fpm.service to /usr/lib/systemd/system/php-fpm.service.      
[root@localhost php-fpm]# systemctl enable nginx.service 
Created symlink from /etc/systemd/system/multi-user.target.wants/nginx.service to /usr/lib/systemd/system/nginx.service.

wordpress下载安装

官网下载：https://cn.wordpress.org/download/

下载

[root@localhost html]# pwd
/usr/share/nginx/html
[root@localhost html]# ls
50x.html  index.html  phpinfo.php
[root@localhost html]# wget https://cn.wordpress.org/latest-zh_CN.tar.gz

解压

[root@localhost html]# tar -zxvf latest-zh_CN.tar.gz

将wordpress目录下的文件全部复制复制到当前目录

[root@localhost html]# cp -rf wordpress/* .

修改nginx配置文件，将80端口的默认主页设置成index.php

[root@localhost html]# vim /etc/nginx/conf.d/default.conf

重启nginx服务

[root@localhost php-fpm]# systemctl restart nginx.service

使用其他机器直接访问当前机器的ip

出现以下画面表示已经安装好wordpress

根据上面的提示需要一个wp-config.php配置文件

找到一个叫wp-config-sample.php的文件，将其再复制一份命名为wp-config.php

[root@localhost html]# pwd
/usr/share/nginx/html
[root@localhost html]# mv wp-config-sample.php wp-config.php
[root@localhost html]# vim wp-config.php
