基于Java探针和字节码框架来动态修改代码运行时行为

 2019-12-01   作者: TanDi

基本介绍

是什么？

  • 所谓”Java探针“其实就是指Java Agent，一种能够直接与 JVM 交互的代理
  • Java Agent能与JVM交互的前提是JVM本身支持JVM TI（JVM Tool Interface）。这样说的原因是并非所有JVM都支持JVM TI，即JVM TI并不是规范
  • JVM TI 支持需要访问JVM状态的全部工具，包括但不限于：性能分析，Debug调试，监控，线程分析和覆盖率分析等等。从这点可以看出 JVM TI 是很强大的
  • Java从JDK5开始提供了实现Java Agent的API，位于java.lang.instrument包下

Java Agent与 JVM TI 的关系

  • Java Agent通过实现JVM TI的回调方法来获取或操作 JVM 的相关性信息

  • Java Agent可以简单理解为是一段需要依附在主程序上的代码。目前的依附方式有两种（都属于本机内操作）

    • 启动依附：即在启动主程序时指定Java Agent -javaagent:代理.jar[=参数]

    • 使用 Attach API 依附：该API位于 tool.jar 的com.sun.tools.attach包下。该方式可以实现动态依附，既即使主程序正在运行也可以操作

可能经过上面一番说明你依然感觉很困惑，但没关系，我们来看看Java Agent 的身影

  • 热部署插件JRebel
  • 懒人开发包Lombok
  • 分布式链路追中服务的Java探针（例如SkyWalking和Pinpoint）
  • 安全(不能改内容)且动态的Java跟踪工具 BTrace
  • 阿里开源的在线分析诊断工具 Arthas
  • JDK自带的诊断工具，例如 Visual VM、JConsole

Java Agent实现

了解基本API

经过一番了解过后我想大家对 Java Agent 的情况已经略知一二了。接下来以案例的形式对 Java Agent 开发进行快速入门。案例分为两部分：

  • 第1部分，以-javaagent参数进行依附，模拟在不修改主程序源码情况下为目标类增加方法
  • 第2部分，以Attach API进行依附，模拟主程序处于运行时，为目标织入环绕代码（类似于Spring AOP中的 @Around 功能）

在这之前先了解一下 Java Agent 的 jar 包结构：

  • Agent class

    实现了 JVM TI 回调方法的代理类，相当于普通程序的入口类(Main)。其中有两个回调方法可以实现，分别是：premain 和 agentmain，各自为 -javaagent依附和Attach API依附的入口方法

  • Packaging

    即包结构，如图所示需要创建 META-INF/MANIFEST.MF文件结构。其中 MANIFEST.MF 包含以下描述配置

    • Agent-Class：实现了agentmain回调方法的入口类限定名
    • Premain-Class：实现了premain 回调方法的入口类限定名
    • Boot-Class-Path：依赖的路径，可以省略但需要将依赖一同编译进jar包
    • Can-Redefine-Classes：是否允许重新定义类，默认为false
    • Can-Retransform-Classes：是否允许替换类的字节码，默认为false

Java Agent实现

public class Agent {
    /**
     * 启动依附回调方法
     * @param agentArgs
     * @param inst
     */
    public static void premain(String agentArgs, Instrumentation inst) {
        System.out.println("========================= into premain =========================");
        System.out.println("参数：" + agentArgs);
        inst.addTransformer(new PersonClassFileTransformer(agentArgs), true);
    }

    /**
     * Attach API依附回调方法
     * @param agentArgs
     * @param inst
     */
    public static void agentmain(String agentArgs, Instrumentation inst) {
        premain(agentArgs, inst);
    }
}


class PersonClassFileTransformer implements ClassFileTransformer {
    private String targetClassName;

    PersonClassFileTransformer(String targetClassName) {
        this.targetClassName = targetClassName;
    }

    @Override
    public byte[] transform(ClassLoader loader, String className, Class<?> classBeingRedefined,
                            ProtectionDomain protectionDomain, byte[] classfileBuffer) throws IllegalClassFormatException {
        // 修改类的限定名格式
        String target = className.replace("/", ".");

        if (targetClassName.equals(target)) {
            try {
                // 获取目标类
                ClassPool classPool = ClassPool.getDefault();
                CtClass ctClass = classPool.makeClass(new ByteArrayInputStream(classfileBuffer));

                // 构建一个新的方法，案例以toString为例
                // 如果不重写toString，默认对象会用Object的toString，输出类容是限定名加十六进制地址
                // 这里我们重载toString让其输出Person的name
                final String method = new StringBuilder()
                        .append("public String toString() { ")
                        .append("return \"name=\" + this.name; }")
                        .toString();
                CtMethod make = CtNewMethod.make(method, ctClass);

                // 将添加到类中（注意：如果是修改操作无需进行addMethod）
                ctClass.addMethod(make);
                return ctClass.toBytecode();
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
        return null;
    }
}

Java Agent实现

Java Agent实现（依赖和上面案例一样不变）

public class Agent {

    /**
     * 启动依附回调方法
     * @param agentArgs
     * @param inst
     */
    public static void premain(String agentArgs, Instrumentation inst) {
        agentmain(agentArgs, inst)
    }

    /**
     * Attach API依附回调方法
     * @param agentArgs
     * @param inst
     */
    public static void agentmain(String agentArgs, Instrumentation inst) {
        System.out.println("========================= into agentmain =========================");
        System.out.println("参数：" + agentArgs);

        // 获取所有已加载的类
        Class[] allLoadedClasses = inst.getAllLoadedClasses();
        for (int i = 0; i < allLoadedClasses.length; i++) {
            Class clazz = allLoadedClasses[i];
            // 只往目标类织入代码
            if (clazz.getName().equals(agentArgs)) {
                try {
                    // 添加转换器
                    inst.addTransformer(new PersonClassFileTransformer(agentArgs), true);
                    // 用转换器重新转换类
                    inst.retransformClasses(clazz);
                } catch (Exception e) {
                    e.printStackTrace();
                }
            }
        }
    }

}

class PersonClassFileTransformer implements ClassFileTransformer {
    private String targetClassName;

    PersonClassFileTransformer(String targetClassName) {
        this.targetClassName = targetClassName;
    }

    @Override
    public byte[] transform(ClassLoader loader, String className, Class<?> classBeingRedefined,
                            ProtectionDomain protectionDomain, byte[] classfileBuffer) throws IllegalClassFormatException {
        String target = className.replace("/", ".");
        if (targetClassName.equals(target)) {
            try {
                // 获取到类
                ClassPool classPool = ClassPool.getDefault();
                CtClass ctClass = classPool.get(targetClassName);
                // 获取到目标类的sayHi方法
                CtMethod method = ctClass.getDeclaredMethod("sayHi");
                // 在say之前和之后织入环绕代码
                method.insertBefore("{ System.out.println(\"----- agent before -----\"); }");
                method.insertAfter("{ System.out.println(\"----- agent after -----\"); }");
                // 返回类的最终字节流
                return ctClass.toBytecode();
            } catch (Exception e) {
                e.printStackTrace();
            }
        }
        return null;
    }
}

测试类（Attach API操作）

public class AgentTest {
    public static void main(String[] args) throws Exception {
        // 遍历所有vm进程来获取目标进程（代替手动jps查找）
        List<VirtualMachineDescriptor> vms = VirtualMachine.list();
        for (int i = 0; i < vms.size(); i++) {
            VirtualMachineDescriptor descriptor = vms.get(i);
            // 要求主程序为wiki.td.Main才进行attach
            if ("wiki.td.Main".equals(descriptor.displayName())) {
                // 往目标attach agent
                VirtualMachine attach = VirtualMachine.attach(descriptor.id());
                // 参数1：agent jar, 参数2：agent参数
                attach.loadAgent("E:/ProgrammingEnvironment/workspace/IdeaProjects/agent/target/agent-1.0-jar-with-dependencies.jar", "wiki.td.Person");
                TimeUnit.SECONDS.sleep(20);
                attach.detach(); // 20s后detach agent
            }
        }
    }
}
